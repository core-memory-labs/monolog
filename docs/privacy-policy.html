1. Общие положения

Настоящая Политика конфиденциальности описывает, как приложение Monolog обрабатывает данные пользователей.

Главное: Monolog — это локальный приватный блокнот, который хранит все данные исключительно на устройстве пользователя. Приложение не собирает, не передаёт и не хранит персональные данные на внешних серверах.

2. Какие данные обрабатывает приложение

2.1. Данные, создаваемые пользователем

Приложение хранит следующие данные, которые создаёт сам пользователь:

  • Текстовые записи (заметки)
  • Топики (категории для организации записей)
  • Прикреплённые файлы и изображения
  • Настройки приложения (выбор темы оформления)

Все эти данные хранятся локально на устройстве пользователя в защищённой директории приложения и не передаются третьим лицам.

2.2. Данные, получаемые из других приложений

При использовании функции «Поделиться» (Share Intent) приложение может получать текст, изображения, файлы и URL-адреса из других приложений. Эти данные сохраняются локально в записях пользователя.

2.3. Сетевые запросы

Приложение выполняет сетевые запросы только для загрузки превью ссылок (OpenGraph-метаданные):

  • Запрашиваются только публично доступные веб-страницы
  • Загружаются только заголовок, описание и изображение превью
  • Данные кэшируются локально на устройстве
  • Никакие персональные данные пользователя не передаются

3. Какие данные НЕ собирает приложение

Приложение не собирает и не передаёт:

  • Персональные данные (имя, email, телефон)
  • Данные геолокации
  • Идентификаторы устройства
  • Данные об использовании приложения (аналитика)
  • Рекламные идентификаторы
  • Информацию о контактах или календаре

4. Хранение данных

Все данные хранятся в локальной базе данных SQLite в защищённой директории приложения. Доступ к этим данным имеет только само приложение.

Пользователь может экспортировать свои данные в ZIP-архив. При удалении приложения все данные удаляются автоматически.

5. Разрешения приложения

Разрешение Назначение
Камера Съёмка фото для прикрепления к записям
Доступ к файлам/фото Выбор изображений и файлов из галереи
Интернет Загрузка превью ссылок (OpenGraph)

6. Передача данных третьим лицам

Приложение не передаёт данные пользователей третьим лицам, рекламным сетям, аналитическим сервисам или любым другим внешним службам.

7. Права пользователя

Пользователь имеет полный контроль над своими данными:

  • Доступ: все данные доступны внутри приложения
  • Экспорт: функция экспорта в ZIP-архив
  • Удаление: можно удалить любые записи или всё приложение
  • Изменение: можно редактировать любые записи
